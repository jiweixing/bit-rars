Cool Capability: Plug-in Tools

Old tools in MARS 4.5 are selected and modified for usage in Risc-V. Just like MAR 4.5, RARS 0.1 is capable of running externally-developed software that meets a set of requirements. Everyone is encouraged to contribute new tools to this system. The requirements for such a program are:
  1. It implements the rars.tools.MarsTool interface.
  2. It is part of the rars.tools package.
  3. It compiles cleanly into a ".class" file, stored in the rars/tools directory.
RARS will detect all qualifying tools upon startup and include them in its Tools menu. When a tool's menu item is selected, an instance of it will be created using its no-argument constructor and its action() method will be invoked. If no qualifying tools are found at RARS startup, the Tools menu will not appear.
