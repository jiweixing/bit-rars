Introduction

BIT-RARS is a work base on MARS(http://www.cs.missouristate.edu/MARS/).

BIT stands for Beijing Institute of Technology

RARS, the Risc-V Assembler and Runtime Simulator, will assemble and simulate the execution of Risc-V assembly language programs. It can be used through its integrated development environment (IDE). RARS is written in Java and requires at least Release 1.6 of the J2SE Java Runtime Environment (JRE) to work. It is distributed as an executable JAR file. The source code can be found at https://github.com/jiweixing/bit-rars.

As of Release 0.1, RARS assembles and simulates over 30 basic instructions of the RV32I instruction set, few pseudo-instructions or instruction variations, several syscall functions mainly for console, random number generation and more. These are listed in separate help tabs.

When writing RARS, the reference is "riscv-spec-v2.2" and "https://github.com/riscv". Although work has been done according to the standard format and usage of Risc-V as much as possible, we still have to compromise a lot for our work is based on MARS which is a tool for MIPS.
